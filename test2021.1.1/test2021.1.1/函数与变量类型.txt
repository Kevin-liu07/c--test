#include<stdio.h>
int a = 20;//全局变量
void func2()
{
	static int i = 3;//静态局部变量
	int k = 0;//局部变量
	i++; k++;
	printf("func2 i=%d,k=%d,a=%d \n", i, k, a);
	a++;
}
int main()
{
	a = 20;
	func2();
	func2();
	return 0;
}

变量主要可以分为:全局变量、局部变量、静态局部变量

  按生存期区别，全局变量、静态局部变量和程序的生存期一样，而局部变量只有定义它的函数被执行时才出生。

  按程序员可用区域分，全局变量定义后，所有的函数都可以使用;局部和静态局部变量只在定义它的函数内可用，

  静态局部变量由于生存期和程序一致，所以函数返回后，该变量的值不会消失；而局部变量，但是函数返回后所有内容失效.

  全局变量和静态变量如果没有手工初始化，则由编译器初始化为0,且静态局部只初始化一次。局部变量的值不可知，每次都重新初始化。
